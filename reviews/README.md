# Reviews Display Web App

This project is a simple web application that showcases customer reviews dynamically. It allows users to navigate through reviews using navigation buttons or view a random review with a surprise button.

## Table of Contents

- [Introduction](#introduction)
- [Features](#features)
- [Usage](#usage)

## Introduction

The web application is designed to display customer reviews dynamically. It includes functionalities to view reviews sequentially or access a random review. The reviews are sourced from local data stored within the JavaScript file.

## Features

- Sequential navigation through reviews using "Previous" and "Next" buttons.
- Display of a random review using the "Surprise Me" button.
- Responsive design for seamless viewing on various devices.

## Usage

To use this web application, open the `index.html` file in a web browser. The interface will display a review along with the reviewer's name, job title, and an image. The "Previous" and "Next" buttons allow navigation between reviews, while the "Surprise Me" button displays a random review.
